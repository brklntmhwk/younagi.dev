## Tables

#### Syntax

```markdown
| Italics   | Bold     | Code   | Thead |
| --------- | -------- | ------ | ----- |
| _italics_ | **bold** | `code` | TBody |
```

#### Output

| Italics   | Bold     | Code   | Very very very long Thead addEventListener(type, listener) |
| --------- | -------- | ------ | ---------------------------------------------------------- |
| _italics_ | **bold** | `code` | Very very very long TBody addEventListener(type, listener) |

## List Types

### Ordered List

#### Syntax

```markdown
1. First item
2. Second item
3. Third item
```

#### Output

1. First item
2. Second item
3. Third item

### Unordered List

#### Syntax

```markdown
- List item
- Another item
- And another item
```

#### Output

- List item
- Another item
- And another item

### Nested list

#### Syntax

```markdown
- Fruit
  - Apple
  - Orange
  - Banana
- Dairy
  - Milk
  - Cheese
```

#### Output

- Fruit
  - Apple
  - Orange
  - Banana
- Dairy
  - Milk
  - Cheese
